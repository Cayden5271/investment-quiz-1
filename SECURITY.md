# Security Policy

## Supported Versions

We release patches for security vulnerabilities. Which versions are eligible for receiving such patches depends on the CVSS v3.0 Rating:

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue. Instead, please report it via one of the following methods:

1. **Email**: [Your email address] (if you have one set up)
2. **GitHub Security Advisory**: Use GitHub's private vulnerability reporting feature

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and work with you to address the issue.

## Security Considerations

This application:
- Stores data locally in the browser (localStorage)
- Does not send data to external servers
- Does not collect or store personal information
- Is client-side only

However, users should be aware that:
- Data stored in localStorage is accessible to any script on the domain
- Users should not enter sensitive financial information
- This is an educational tool, not a secure financial platform

## Disclaimer

This tool is for **educational purposes only** and is **not financial advice**. Users should not rely on this tool for actual financial planning without consulting a qualified professional.
